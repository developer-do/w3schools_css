CSS Layout max-width

너비, 최대 너비 및 여백 사용 : 자동

이전 장에서 언급했듯이 block 요소는 항상 사용 가능한 전체 너비를 사용합니다. (가능한 한 외쪽과 오른쪽으로 뻗어있음)

block 요소의 width를 설정하면 해당 요소의 가장 자리까지 확장되지 않는다.

그런다음 여백을 자동으로 설정하여 요소 중심을 block 안에 수평으로 맞출 수 있다. 요소는 지정된 너비를 차지하며, 나머지 공간은 두 여백 사이에서 균등하게 분할된다.

이 div 요소의 너비는 500px 이고 여백은 auto로 설정됩니다.

참고: 위의 문제는 브라우저 윈도우가 div 요소 너비보다 작은 경우 발생합니다. 그런 다음 브라우저는 가로 스크롤 막대를 페이지에 추가합니다.

대신에 max-width를 사용하는 것은 이 상황에서, 작은 창들에 대한 브라우저의 처리를 향상 시킬 것이다. 이는 소규모 기기에서 사이트를 사용할 수 있도록 할 때 중요하다.

이 div 요소의 최대 너비는 500px 이고 여백은 auto로 설정됩니다.

브라우저 창 크기를 500px 미만으로 조정하여 두 div의 차이점을 확인하십시오.
