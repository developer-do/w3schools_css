CSS Outline

CSS outline

윤곽은 요소를 "눈에 띄게" 만들기 위해 테두리를 제외한 요소 주위에 그려지는 선입니다.

border 바로 바깥에 위치한다.

CSS에는 다음과 같은 outline 속성이 있습니다.

  • outline-style
  • outline-color
  • outline-width
  • outline-offset
  • outline

Tip: outline은 border와 다릅니다. 테두리와는 달리, outline은 요소의 border 외부로 그려지며 다른 내용과 겹칠 수 있습니다.

또한 outline은 치수의 일부가 아니다. outline의 전체 폭과 높이는 외곽선 너비의 영향을 받지 않는다.

outline style

이 outline-style 속성은 개요의 스타일을 지정하며 다음 값 중 하나를 가질 수 있습니다.

  • dotted - 점선으로 된 윤곽을 정의합니다.
  • dashed - 파선 테두리를 정의합니다.
  • solid - 단색 테두리를 정의합니다.
  • double - 이중 윤곽을 정의합니다.
  • groove - 3D 그루브 아웃 라인을 정의합니다.
  • ridge - 3D ridged outline을 정의합니다.
  • inset - 3D 인세 트 개요 정의
  • outset - 3D 시작 개요 정의
  • none - 윤곽을 정의하지 않습니다.
  • hidden - 숨겨진 개요 정의

Tip: outline-style 속성이 설정되지 않은 한 다른 outline속성은 아무런 영향을 미치지 않습니다.

윤곽선 색

이 outline-color 속성은 외곽선의 색상을 설정하는 데 사용됩니다.

색상은

  • name - "red"와 같은 색상이름 사용가능
  • RGB - RGB 값을 지정한다. rgb(0,0,0)
  • 16진수 - 16진수와 같이 지정 가능. #ffff00
  • invert - 색상 반전을 수행합니다. (색상 배경에 관계없이 외곽선을 볼 수 있음)

outline width

이 outline-width 속성은 외곽선의 너비를 지정하며 다음 값 중 하나를 가질 수 있습니다.

  • 얇은 (일반적으로 1px)
  • 중간 (일반적으로 3px)
  • 두꺼운 (일반적으로 5px)
  • 특정 크기 (px, pt, cm, em 등)

outline - 속기 속성

이 outline 속성은 다음과 같은 개별 개요 속성을 설정하기 위한 속기 속성입니다.

  • outline-width
  • outline-style(필수)
  • outline-color

꿀 Tip: 이 outline 속성에 속기 속성을 사용할 때 순서가 필요 없습니다.
