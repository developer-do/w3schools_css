CSS Tables

CSS를 사용하면 HTML표의 모양을 크게 향상시킬 수 있습니다.

표 테두리

CSS에서 표 테두리를 지정하려면 border속성을 사용하십시오.

이름 성
베드로 그리핀
로이스 그리핀 
    table, th, td {
      border: 1px solid black;
    }

위 예제의 표에는 이중 테두리가 있습니다. 이는 테이블과 th, td요소가 별도의 테두리를 가지고 있기 때문입니다.

표 축소 테두리

이 border-collapse 속성은 테이블 테두리를 단일 테두리로 축소할지 여부를 설정합니다.

이름 성
베드로 그리핀
로이스 그리핀 
    table, th, td {
      border: 1px solid black;
    }
    table {
      border-collapse: collapse;
    }

표 주위의 테두리만 border가 있으면 table element에다가만 border 속성을 지정하면 됩니다.

    table {
      border: 1px solid black;
    }

표 너비와 높이

표의 너비와 높이는 width 및 height 속성에 의해 정의됩니다.

아래 예제는 표의 너비를 100%로 설정하고 th 요소의 높이를 50px로 설정합니다.

이름 성 저금
베드로 그리핀 $100
로이스 그리핀 $150
조 스완 손 $300 
    table {
      width: 100%;
    }
    th {
      height: 50px;
    }

수평 정렬

이 text-align 속성은 th, td 에 내용의 가로 맞춤 (왼쪽, 오른쪽 또는 가운데)을 설정합니다.

기본적으로 th요소의 내용은 가운데 맞춤으로 정렬됩니다.

기본적으로 td요소의 내용은 왼쪽 맞춤으로 정렬됩니다.

이름 성 저금
베드로 그리핀 $100
로이스 그리핀 $150
조 스완 손 $300 
    th {
      text-align: left;
    }

수직 정렬

이 vertical-align속성은 th, td에 내용의 세로 정렬 (위, 아래 또는 가운데)을 설정합니다.

기본적으로 표의 내용은 중간에 정렬됩니다. (th, td 요소 모두 해당)

다음 예제는 td요소의 텍스트 정렬을 아래쪽으로 설정합니다.

이름 성 저금
베드로 그리핀 $100
로이스 그리핀 $150
조 스완 손 $300 
    td {
      height: 50px;
      vertical-align: bottom;
    }

테이블 패딩

표의 테두리와 내용 사이의 간격을 제어하려면 th, td요소에 padding속성을 적용합니다.

이름 성 저금
베드로 그리핀 $100
로이스 그리핀 $150
조 스완 손 $300 
    th, td {
      text-align: left;
      padding: 15px;
    }

수평 분할자

이름 성 저금
베드로 그리핀 $100
로이스 그리핀 $150
조 스완 손 $300

수평 분할자의 경우 th, td요소에 border-bottom 속성을 추가해주십시오.

    th, td {
      text-align: left;
      border-bottom: 1px solid #ddd;
      padding: 15px;
    }

Hover 테이블

tr에 마우스를 올려놓을때 테이블행을 강조하고 싶으면 :hover속성을 사용하십시오.

이름 성 저금
베드로 그리핀 $100
로이스 그리핀 $150
조 스완 손 $300 
    tr:hover {
      background-color: #f5f5f5;
    }

줄무늬 테이블

이름 성 저금
베드로 그리핀 $100
로이스 그리핀 $150
조 스완 손 $300

제브라 스트라이브 테이블의 경우 nth-child()선택기를 사용하여 모든 짝수(또는 홀수) 테이블 행에 background-color 속성을 추가합니다.

    tr:nth-child(even) {
      background-color: #f2f2f2;
    }

표 색상

아래 예제는 th 요소의 배경색과 텍스트 색상을 지원합니다.

이름 성 저금
베드로 그리핀 $100
로이스 그리핀 $150
조 스완 손 $300 
    th {
      background-color: #4CAF50;
      color: white;
    }

반응형 테이블

반응형 테이블은화면이 너무 작아서 전체 내용을 표시할 수 없는 경우 수평 스크롤 막대를 표시합니다.

div와 같은 컨테이너로 table 요소를 감싸고 div에 overflow-x: auto속성을 적용합니다.
