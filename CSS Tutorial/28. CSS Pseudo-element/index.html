CSS Pseudo-elements

CSS Pseudo-element란 무엇인가?

CSS 가상 element는 element의 지정된 부분을 스타일 화 하는데 사용됩니다.

예를 들어 다음과 같이 사용할 수 있습니다.

  • 요소의 첫 번째 문자 또는 선 스타일 지정
  • 요소의 내용 앞이나 뒤에 내용 삽입

syntax

문법

    selector::pseudo-element {
      property: value;
    }

이중 콜론 표기법에 유의하십시오. - ::first-line vs :first-line

이중 콜론은 CSS3의 가상 요소에 대한 단일 클론 표기법을 대체했습니다. 이것은 가상 클래스와 가상 요소를 구별하기 위한 W3C의 시도였습니다.

단일 콜론 구문은 CSS2 및 CSS1의 가상 클래스 및 가상 요소에 모둔 사용되었습니다.

이전 버전과의 호환성을 위해 단일 콜론 구문은 CSS2 및 CSS1 가상 요소에 허용됩니다.

::first-line 가상 요소

::first-line 가상 요소는 텍스트의 첫번째 줄에 특별한 스타일을 추가하는 데 사용됩니다.

    p::first-line {
      color: #ff0000;
      font-variant: small-caps;
    }

참고: ::first-line 가상 요소는 block 요소에만 적용될 수 있습니다.

다음 속성이 ::first-line 의사 요소에 적용됩니다.

  • font 속성
  • color 속성
  • background 속성
  • word-spacing 속성
  • letter-spacing 속성
  • text-decoration 속성
  • vertical-align 속성
  • text-transform 속성
  • line-height 속성
  • clear 속성

::first-letter 가상 요소

::first-letter 가상 요소는 텍스트의 첫 글자에 특별한 스타일을 추가하는 데 사용됩니다.

    p::first-letter {
      color: #ff0000;
      font-size: xx-large;
    }

참고: ::frist-letter 가상 요소는 ::first-line 처럼 블록 요소에만 가능합니다.

다음 속성은 ::first-letter 가상 요소에 적용됩니다.

  • font 속성
  • color 속성
  • background 속성
  • margin 속성
  • padding 속성
  • border 속성
  • text-decoration 속성
  • vertical-align 속성
  • text-transform 속성
  • line-height 속성
  • float 속성
  • clear 속성

가상 요소와 css 클래스

가상 요소는 css 클래스와 결합 할 수 있습니다.

    p.intro::first-letter {
      color: #ff0000;
      font-size: 200%;
    }

다중 가상 element

여러개의 가상 요소를 결합 할 수도 있습니다.

    p::first-letter {
      color: #ff0000;
      font-size: xx-large;
    }
    p::first-line {
      color: #0000ff;
      font-variant: small-caps;
    }

CSS - ::before 가상 element

::before 는 기존 element이전에 내용을 삽입 할 수 있습니다.

CSS - ::after 가상 element

::after 는 기존 element이후에 내용을 삽입 할 수 있습니다.

CSS - ::selection 가상 elemnet

::selection 는 사용자에 의해 선택되는 (드래그 되는) 부분이다.

다음 ::selection에는 color, background, cursor과 outline 들의 속성을 적용할 수 있다.

    ::selection {
      color: red;
      background-color: yellow;
    }
