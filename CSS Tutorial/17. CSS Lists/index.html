CSS Lists

HTML 목록 및 CSS 목록 속성

HTML에는 두가지 주요 유형이 있다.

  1. 순서가 없는 목록 ul - 목록 항목은 글머리 기호로 표시됨
  2. 순서가 있는 목록 ol - 목록 항목에 숫자나 문자가 표시됨

CSS 목록 속성을 사용하면 다음을 수행할 수 있습니다.

  1. 정렬된 목록에 대해 다른 목록 항목 마커 설정
  2. 정렬되지 않은 목록에 대해 다른 목록 항목 마커 설정
  3. 이미지를 목록 항목 마커로 설정합니다.
  4. 목록 및 배경 항목에 배경색 추가

하지만 ul, li를 거의 list-style: none으로 놓고 사용하기 때문에 기본 CSS 속성들만 보고 넘어가겠다.

다른 목록 항목 타입

이 list-style-type 속성은 목록 항목표시의 유형을 지정합니다.

    ul.a {
      list-style-type: circle;
    }
    
    ul.b {
      list-style-type: square;
    }
    
    ol.c {
      list-style-type: upper-roman;
    }
    
    ol.d {
      list-style-type: lower-alpha;
    }

값 중 일부는 정렬되지 않은 목록에 대한 값이고 일부는 정렬된 목록에 대한 값입니다.

목록 항목 타입의로서의 img

이 list-style-image 속성은 이미지를 목록 항목 마커로 지정합니다.

    ul {
      list-style-image: url ('sqpurple.gif');
    }

list item marker 위치 지정

이 list-style-position 속성은 목록 항목 마커의 위치를 지정합니다.

    ul.a {
      list-style-position: outside;
    }
    
글머리 기호가 목록 항목 외부에 있음을 뜻함

    ul.b {
      list-style-position: inside;
    }
    
글머리 기호가 목록 항목 내부에 있음을 뜻함

기본 설정 제거

list-style-type:none 속성은 마커나 순서를 제거해준다. 목록에는 기본 여백 및 채우기도 있기때문에 margin: 0; padding: 0;을 주면 된다.

list 속기속성

색상을 사용하여 목록을 스타일링 할 수 있다.
