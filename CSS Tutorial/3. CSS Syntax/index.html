CSS Syntax and selectors

CSS 구문

CSS 규칙 집합은 선택기와 선언 블록으로 구성됩니다.

ex)

    p {
      color: red;
      text-align: center;
    }

CSS 셀렉터

CSS 선택기는 HTML 요소를 요소 이름, ID, 클래스, 특성 등에 따라 찾거나 선택하는데 사용됩니다.

요소 선택기

요소 선택기는 요소 이름을 기반으로 요소를 선택합니다.

    p {
      color: red;
      text-align: center;
    }

다음과 같이 페이지의 모든 p 요소를 선택해서 스타일을 지정할 수 있습니다.

id 셀렉터

id selector는 HMTL 요소의 id 속성을 사용하여 특정 요소를 선택합니다.

요소의 id는 페이지 내에서 고유해야 하므로 id selector는 하나의 고유한 요소를 선택하는데 사용됩니다.

특정 ID가 있는 요소를 선택하려면 해시(#) 문자 다음에 요소의 ID를 씁니다.

아래 스타일 규칙은 id = "para1"인 HTML 요소에 적용됩니다.

    #para1 {
      text-align: center;
      color: red;
    }

Tip: ID 이름은 숫자로 시작할 수 없습니다.

Selector 클래스

클래스 선택기는 특정 클래스 속성이 있는 요소를 선택합니다.

특정 클래스가 있는 요소를 선택하려면 마침표 (.) 문자 다음에 클래스 이름을 씁니다.

아래 예제에서 class ="center" 인 모든 HTML 요소는 빨간색으로 중앙 정렬 됩니다.

    .center {
      text-align: center;
      color: red;
    }

특정 HTMl 요소만 클래스의 영향을 받도록 지정할 수도 있습니다.

아래 예제에서는 class="center"인 < p > 요소만 가운데 정렬됩니다.

    p.center {
      text-align:center;
      color: red;
    }

HTML 요소는 둘 이상의 클래스를 참조 할 수도 있습니다.

아래 예제에서 < p > 요소는 class = "center" 및 class = "large"에 따라 스타일이 지정됩니다.

    < p class="center large"> This is paragraph refers to two classes. < /p >

Note: 클래스 이름은 숫자로 시작할 수 없습니다.

그룹화 선택기

다음과 같이 같은 스타일 정의가 있는 요소가 있는 경우

    h1 {
      text-align:center;
      color:red;
    }
    h2 {
      text-align:center;
      color:red;
    }
    p {
      text-align:center;
      color:red;
    }

코드를 최소하 하기 위해 선택기를 그룹화 하는것이 좋습니다.

선택기를 그룹화 하려면 각 선택기를 쉼표로 구분하십시오.

아래 예제에서 위 코드의 선택기를 그룹화 했습니다.

    h1, h2, p {
      text-align: center;
      color: red;
    }
