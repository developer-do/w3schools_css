CSS Layout - float and clear

CSS float 속성은 요소가 어떻게 떠 다니는지를 지정합니다.

CSS clear 속성은 지워진 요소 옆에 어떤 요소가 떠오르며 어떤면에 놓일 수 있는지 지정합니다.

float 속성

이 float 속성은 내용의 위치 지정 및 형식 지정(예: 이미지를 컨테이너의 텍스트 왼쪽으로 띄우기)에 사용됩니다.

이 float 속성은 다음 값 중 하나를 가질 수 있습니다.

  • left - element가 컨테이너의 왼쪽으로 위치됩니다.
  • right - element가 컨테이너의 오른쪽으로 위치합니다.
  • none - element가 float되지 않음 (텍스트에서 나타나는 바로 그 위치에 표시됨.) 기본값
  • inherit - element가 상위 요소의 float을 상속한다.

clear 속성

이 clear 속성은 지워진 요소 옆에 어떤 요소가 떠오르며 어떤면에 올 수 있는지 지정합니다.

이 clear 속성은 다음 값 중 하나를 가질 수 있습니다.

  • none - 양쪽에 float 요소를 허용합니다. 기본값.
  • left - 왼쪽에 부동 요소가 허용되지 않음
  • right - 오른쪽에 부동 요소가 허용되지 않음
  • both - 왼쪽 또는 오른쪽에 부동 요소가 허용되지 않음
  • inherit - element은 부모의 명확한 값을 상속받습니다.

이 clear 속성을 사용하는 가장 일반적인 방법은 요소에 float 속성을 사용한 후 입니다.

float을 제거할 때 float에 맞춰야 합니다. 요소가 왼쪽으로 떠오르면 왼쪽으로 clear하십시오.

float되 있는 element은 계속 떠있지만 지워진 element은 그 아래 웹페이지에 나타날 것입니다.

    div {
      clear: left;
    }

clearfix 해킹

element가 포함된 element보다 크고 float된 경우 해당 element가 컨테이너 외부에서 overflow 됩니다.

그런 다음 overflow: auto로 element를 포함하여야 이 문제를 해결할 수 있습니다.

    .clearfix {
      overflow: auto;
    }
    .clearfix:after {
      content: "";
      display: block;
      clear: both;
    }

box-sizing란 무었입니까 ?
세 개의 플로팅 상자를 나란히 쉽게 만들 수 있습니다. 그러나 각 상자의 너비를 확대하는 항목을 추가하면 상자가 끊깁니다. 이 box-sizing속성을사용하면 상자의 전체 너비에 채우기 및 테두리를 포함 할 수 있으며 상자 안쪽에 채우기가 유지되고 중단되지 않도록 할 수 있습니다.
