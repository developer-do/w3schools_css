텍스트 정렬

이 text-align 속성은 텍스트의 가로 맞춤을 설정하는 데 사용됩니다.

텍스트는 왼쪽 또는 오른쪽 정렬, 가운데 맞춤 또는 양쪽 맞춤이 가능합니다.

다음 예제에서는 가운데 정렬 및 왼쪽 및 오른쪽 정렬 텍스트를 보여줍니다.

    h1{
      text-align: center;
    }
    h2 {
      text-align: left;
    }
    h3 {
      text-align: right;
    }

text-align 속성이 "양쪽 맞춤"으로 설정되면 모든 선의 폭이 동일하고 왼쪽 및 오른쪽 여백이 직선형이 되도록 각 선이 늘어납니다. (예: 잡지 신문)

    div {
      text-align: justify;
    }

텍스트 장식

이 text-decoration 속성은 텍스트에서 장식을 설정하거나 제거하느 데 사용됩니다.

이 text-decoration: none;은 종종 링크에서 밑줄을 제거하는 데 사용됩니다

    a {
      text-decoration: none;
    }

다른 text-decoration 값은 텍스트를 장식하는 데 사용됩니다.

    h1 {
      text-decoration: overline;
    }
    h2 {
      text-decoration: line-through;
    }
    h3 {
      text-decoration: underline;
    }

Tip: 링크가 아닌 텍스트에 밑줄을 긋는 것은 독자에게 혼동을 주기 때문에 링크 텍스트가 아닌 텍스트에 밑줄을 권장하지 않습니다.

텍스트 변환

이 text-transform 속성은 텍스트에서 대문자와 소문자를 지정하는 데 사용됩니다.

모든것을 대문자 또는 소문자로 바꾸거나 각 단어의 첫 글자를 대문자로 사용할 수도 있습니다.

    p.uppercase {
      text-transform: uppercase;
    }
    p.lowercase {
      text-transform: lowercase;
    }
    p.capitalize {
      text-transform: capitalize;
    }

텍스트 들여쓰기

이 text-indent속성은 텍스트의 첫 번째 줄에 들여쓰기를 지정하는데 사용됩니다.

In my younger and more vulnerable years my father gave me some advice that I've been turning over in my mind ever since. 'Whenever you feel like criticizing anyone,' he told me, 'just remember that all the people in this world haven't had the advantages that you've had.'

    p {
      text-indent: 50px;
    }
