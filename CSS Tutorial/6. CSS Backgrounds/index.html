CSS 배경

CSS 배경속성은 요소의 배경 효과를 정의하는 데 사용됩니다.

  • 배경색 - background-color
  • 배경 이미지 - background-image
  • 배경 반복 - background-repeat
  • 배경 부착 - background-attachment
  • 배경 위치 - background-position

배경색

이 background-color 속성은 요소의 배경색을 지정합니다.

RGB값 또는 HEX값을 사용할 수 있습니다.

    body {
      background-color: lightblue;
    }

배경 이미지

이 background-image 속성은 요소의 배경으로 사용할 이미지를 지정합니다.

기본적으로 전체 요소를 포함하도록 반복됩니다.

페이지의 배경 이미지는 다음과 같이 설정할 수 있습니다.

    body {
      background-image: url("paper.gif");
    }

참고: 배경 이미지를 사용할 때는 텍스트를 방해하지 않는 이미지를 사용하십시오.

배경 이미지 - 수평 또는 수직 반복

기본적으로 background-image 속성은 이미지를 가로 및 세로로 반복합니다. (default)

일부 이미지는 가로 또는세로로만 반복되거나 다음과 같이 이상하게 보입니다.

참고

위의 이미지가 수평으로만 반ㄴ복되는경우 background-repeat: repeat-x; 배경이 더 좋아보입니다.

참고

Tip: 이미지를 세로로 반복하려면, background-repeat: repeat-y;를 하세요.

배경 이미지 - 위치 및 반복 없음 설정

배경 이미지를 한 번만 표시하는 것은 background-repeat속성에 의해 지정됩니다.

    body {
      background-image: url("img_tree.png");
      background-repeat: no-repeat;
    }

위의 예에서 배경 이미지는 텍스트와 같은 위치에 표시됩니다. 이미지의 위치를 변경하여 텍스트를 너무 많이 방해하지 않도록 하고싶습니다.

이미지의 위치는 다음 background-position 속성에 의해 지정됩니다.

    background-image: url("ima_tree.png");
    background-repeat: no-repeat;
    background-position: right top;

배경 이미지 - 고정 위치

배경 이미지를 고정해야 한다고 지정하려면 (페이지의 나머지 부분과 함께 스크롤 하지 않음) 다음 background-attachment 속성을 사용하십시오.

    body {
      background-image: url("img_tree.png");
      background-repeat: no-repeat;
      background-position: right top;
      background-attachment: fixed;
    }

배경 - 속기 속성

코드 단축을 위해 하나의 단일 속성에서 모든 배경 속성을 지정할 수도 있습니다. 이를 속기 속성이라고 합니다.

배경에 대한 속기속성은 background 다음과 같습니다.

    body {
      background: #ffffff url("img_tree.png") no-repeat right top;
    }
