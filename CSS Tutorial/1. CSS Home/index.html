CSS 듀토리얼

CSS는 HTML 문서의 스타일을 설명하는 언어입니다.

CSS는 HTML 요소를 표시하는 방법을 설명합니다.

이 듀토리얼은 기본에서 고급까지 CSS를 가르쳐 줄 것입니다.

각 장의 예

이 CSS 듀토리얼에는 수백가지 CSS예제가 포함되어 있습니다.

온라인 편집기를 사용하여 CSS를 편집하고 버튼을 클릭하여 결과를 볼 수 있습니다.

    body {
      background-color: lightblue;
    }
    h1 {
      color: white;
      text-align: center;
    }
    p {
      font-family: verdana;
      font-size: 20px;
    }

앞으로는 중요한 내용만 HTML에 담으면 실습에 집중하고 빠르게 습득하여 넘기겟습니다.
