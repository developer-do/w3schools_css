CSS Combinators

CSS 연결자

연결자는 선택자 간의 관계를 설명하는 것입니다.

CSS 연결자에는 둘 이상의 단순 선택기가 포함될 수 있습니다. 간단한 선택자 사이에는 결합자를 포함할 수 있습니다.

CSS에는 4가지 다른 결합자가 있습니다.

  • 자손 선택자(space)
  • 자식 선택자(>)
  • 인접 형제 선택기(+)
  • 일반 형제 선택 도구(~)

자손 선택기

자손 선택자는 지정된 요서의 하위 항목인 모든 요소를 찾습니다.

    div p {
      color: yellow;
    }

div 안에 있는 모든 p element에 color을 yellow로 바꿉니다.

자식 선택기

자식 선택기는 지정된 요소의 자식요소를 찾습니다.

    div > p {
      color: yellow;
    }

div 바로 다음 자식들 중에 p element를 찾아서 color을 yellow로 바꿉니다.

인접한 형제 선택기

인첩한 형제 선택기는 지정된 요소의 인접 형제인 모든 요소를 선택합니다.

형제 요소는 같은 부모 요소를 가져야 하며 "인접 요소"는 "바로 뒤에"오는 element를 의미합니다.

    div + p {
      color: green;
    }

div랑 형제선상에 있는 그 바로 다음의 p element를 찾아서 color를 green으로 바꿉니다.

일반 형제 선택기

일반 형제 선택 도구는 지정된 요소 형제인 모든 요소를 선택합니다.

    p ~ p {
      color: red;
    }

p로 시작 (시작하는 p 요소는 제외한다.) 해서 형제 마지막 p까지 확인후 그 안에 있는 모든 p들의 color을 red로 바꾼다.
