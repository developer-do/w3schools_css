CSS Forms

CSS를 사용하면 HTML form의 모양을 크게 향상 시킬 수 있습니다.

입력 필드 스타일 지정하기

width: 속성을 사용하여 입력 필드의 너비를 결정합니다.

    input {
      width: 100%;
    }

위의 예제는 모든 input 태그에 적용됩니다. 특정 입력 유형에만 스타일을 지정하려는 경우 속성 선택자를 사용할 수 있습니다.

  • input[type=text] - 텍스트 필드만 선택합니다.
  • input[type=password] - 비밀번호 필드만 선택합니다.
  • input[type=number] - 숫자 필드만 선택합니다.

패딩된 input

padding 속성을 사용하여 텍스트 필드 안에 공백을 추가하십시오.

Tip : 서로간에 붙어져 있을때 margin을사용사여 외부에 더 많은 공간을 추가할 수 있습니다.

테두리가 있는 input

이 border 속성을 사용하여 테두리 크기와 색상을 변경 border-radius를 사용하여 모서리를 둥글게 만들 수 있습니다.

컬러 input

이 background-color 속성을 사용하여 입력에 배경색을 추가 할 수 있습니다. color로 텍스트 색상도 바꿀 수 있습니다.

집중되는 input

기본적으로 일부 브라우저는 포커스를 클릭하면 입력 주위에 파란색 윤곽선을 추가합니다. outline: none으로 동작을 제거할 수 있습니다.

:focus 설렉터를 사용하여 포커스를 얻으면 입력 필드를 사용하여 무언가를 하십시오.

아이콘 이미지 input

input안에 아이콘은 background-image속성을 사용하여 background-position 속성과 함께 배치 될수 있습니다. 또한 아이콘의 공간을 확보하기위해 padding-left 값을 줍니다.

애니메이션 검색 입력

이 예제에서는 CSS transition 속성을 사용하여 검색 입력이 포커스를 얻을 때 애니메이션의 너비를 애니메이션으로 표시합니다.

스타일 지정 텍스트 영역

Tip: resize 속성을 사용하여 텍스트 영역의 크기가 조절되지 않도록 하십시오.
