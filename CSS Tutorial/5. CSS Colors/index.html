CSS 색상

색상은 미리 정의된 색상 이름 또는 RGB, HEX, HSL, RGBA, HSLA 값을 사용하여 지정됩니다.

색상 이름

HTML에서는 색상 이름을 사용하여 색상을 지정할 수 있습니다.

HTML은 140개의 표준 색상 이름을 지원합니다.

배경색

HTML 요소의 배경색을 설정할 수 있습니다.

    h1 {
      background-color: DodgerBlue;
    }
    p {
      background-color: Tomato;
    }

텍스트 색상

텍스트 색상을설정할 수 있습니다.

    p {
      color: red;
    }
    span {
      color: blue;
    }

테두리 색

테두리 색상을 설정할 수 있습니다.

    h1 {
      border: 2px solid Tomato;
    }
    p {
      border: 2px solid DodgerBlue;
    }
    span {
      border: 2px solid Violet;
    }

색상 값

HTML에서는 RGB값, HEX값, HSL값, RGBA값 및 HSLA 값을 사용하여 색상을 지정할 수도 있습니다.

색상이름 토마토와 동일함

    p {
      background-color: rgb(255,99,71);
      background-color: #ff6347;
      background-color: hsl(9, 100%, 64%);
    }

색 이름 "토마토"와 동일하지만 50% 투명

    p {
      background-color: rgba(255, 99, 71, 0.5);
      background-color: hsla(9m 100%, 64%, 0.5);
    }

RGB값

HTML에서는 다음 공식을 사용하여 색상을 RGB 값으로 지정할 수 있습니다.

rgb(빨강, 초록, 파랑)

각 매개변수 (빨강, 녹색, 파랑)는 0에서 255사이의 색상 강도를 정의합니다.

예를 들어 빨간색은 가장 높은값 (255)으로 설저되고 다른 값은 (0)으로 설정되므로 rgb(255,0,0)는 빨간색으로 표시됩니다.

검은색은 rgb(0,0,0) 흰색은 (255,255,255)

나머지는 딱히 사용하지 않음. rgba로도 충분하다. 그리고 크롬개발자 도구에서도 변경할 수 있으니 이걸로 마치겠다.
