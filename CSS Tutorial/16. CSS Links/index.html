CSS Links

CSS 를 사용하면 링크를 다른 방식으로 스타일을 지정할 수 있습니다.

link style

링크는 어떤 CSS 속성이든 정할 수 있습니다.

    a {
      color: hotpink;
    }

또한 링크의 상태에 따라 다르게 스타일을 지정할 수 있습니다.

  • a:link - 방문하지 않은 링크
  • a:visited - 사용자가 방문한 링크
  • a:hover - 사용자가 마우스를 올려놓을때의 링크
  • a:active - 클릭 한 순간의 링크

여러 링크 상태에 대한 스타일을 설정할 때 몇가지 순서 규칙이 있습니다.

1. a:hover는 a:link 와 a:visited 뒤에 와야합니다.

2. a:active는 반드시 a:hover 다음에 와야합니다.

text decoration

이 text-decoration 속성은 주로 링크에서 밑줄을 제거하는 데 사용됩니다.

text background

이 background-color 속성은 링크의 배경색을 지정할 수 있습니다.

고급 - link button

이 예제는 몇가지 CSS 속성을 결합하여 링크를 상자 / 버튼으로 표시하는 고급 예제를 보여줍니다.

    a:link, a:visited {
      background-color: #f44336;
      color: white;
      padding: 14px 25px;
      text-align: center;
      text-decoration: none;
      display: inline-block;
    }

    a:hover, a:active {
      background-color: red;
    }
