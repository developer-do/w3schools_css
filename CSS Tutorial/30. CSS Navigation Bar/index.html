네비게이션 모음

사용하기 쉬운 네비게이션 기능은 모든 웹 사이트에서 중요합니다.

CSS를 사용하면 지루한 HTML 메뉴를 멋진 탐색 모음으로 변형 할 수 있습니다.

네비게이션 모음 = 링크 목록

탐색 모음에는 표준 HTML이 기본으로 필요합니다.

예제에서는 표준HTML 목록에서 네비게이션 모음을 작성합니다.

네비게이션 바는 기본적으로 링크 목록이므로 ul li 요소를 사용하면 완벽하게 이해할 수 있습니다.

    < ul >
      < li >< a href="default.asp">Home< /a >
      < li >< a href="news.asp">News< /a >
      < li >< a href="contact.asp">Contact< /a >
      < li >< a href="about.asp">About< /a >
    < /ul >

수직 네비게이션 바

수직 네비게이션 바를 만들려면 위 코드와 함께 목록안의 a 요소의 스타일을 지정할 수 있습니다.

    li a {
      display: block;
      width: 60px;
    }

display: block은 링크 요소를 블록 요소로 바꾸며 텍스트 뿐만 아니라 전체 링크 영역을 클릭할 수 있게 되고 padding및 margin 값을 지정할 수 있습니다.
