CSS Box Model

The CSS Box Model

모든 HTML 요소는 상자로 간주 할 수 있습니다. CSS에서는 디자인과 레이아웃에 대해 "Box Model" 이라는 용어를 사용합니다.

CSS 상자 모델은 본질적으로 모든 HTML요소를 감싸는 상자입니다.

margin, border, padding, content로 구성 되어있습니다.

  • content - 텍스트 및 이미지가 나타나는 상자 내용
  • padding - content 주위의 영역과 border 사이의 공간을 차지합니다. 패딩은 투명합니다.
  • border - padding 및 content의 주위를 둘러 쌀 테두리
  • margin - border 바깥 영역을 차지합니다.

box model을 사용하면 요소 주위에 border를 추가하고 content 사이에 공백을 정의 할 수 있습니다.

    div {
      width: 300px;
      border: 25px solid green;
      padding: 25px;
      margin: 25px;
    }

요소의 너비와 높이

모든 브라우저에서 요소의 너비와 높이를 올바르게 설정하려면 상자 모델의 작동 방식을 알아야합니다.

중요!: CSS를 사용하여 요소의 너비 및 높이 속성을 설정하면 내용 영역의 너비와 높이를 설정하기만 하면 됩니다. 요소의 전체 크기를 계산하려면 패딩, 테두리 및 여백도 추가해야 합니다.

총 너비가 350px이 되도록 div 요소 스타일을 지정한다고 가정하면

    div {
      width: 320px;
      padding: 10px;
      border: 5px solid gray;
      margin: 0;
    }

계산은 다음과 같습니다.

320px (width) + 20px (왼쪽 + 오른쪽 padding) + 10px (왼쪽 + 오른쪽 border) + 0px (왼쪽 + 오른쪽 margin)

요소의 full width는 다음과 같이 계산되어야 합니다.

full content width = width + margin-left + padding-left + border-left + border-right + padding-right + margin-right

요소의 full height는 다음과 같이 계산되어야 합니다.

full content height = width + margin-top + padding-top + border-top + border-bottom + padding-bottom + margin-bottom
