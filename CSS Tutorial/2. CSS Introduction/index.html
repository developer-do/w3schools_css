CSS 소개

CSS란 무엇인가 ?

  • CSS는 C의 cascading S의 style S의 sheets 입니다.
  • CSS는 HTML 요소를 화면, 종이 또는 다른 매체에 표시하는 방법을 설명한다.
  • CSS는 한번에 여러 웹페이지의 레이아웃을 제어할 수 있으며 많은 작업을 저장합니다.
  • 외부스타일 시트는 CSS 확장명을 가진 CSS파일에 저장됩니다.

CSS 데모 - 하나의 HTML 페이지 - 여러 스타일

하나의 HTML 페이지에 여러 스타일을 정의할수 있다.

왜 CSS를 사용할까 ?

CSS는 디자인, 레이아웃 및 다양한 장치 및 화면 크기에 대한 디스플레이의 변형을 포함하여 웹 페이지의 스타일을 정의하는데 사용됩니다.

CSS로 많은 작업을 절약할 수 있다.

스타일 정의는 일반적으로 외부 .css 파일에 저장합니다.

외부 스타일 시트 파일을 사용하면 하나의 파일만 변경하여 전체 웹 사이트의 모양을 변경할 수 있습니다.
