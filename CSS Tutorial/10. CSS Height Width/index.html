CSS Height And Width

높이 및 너비 설정

height 및 width 특성 요소의 높이와 폭을 설정하는데 사용된다.

height 와 width는 자동으로 설정할 수 있다.(기본값) 브라우저가 높이와 너비를 계산하거나 px, cm 등과 같은 길이 값 또는 포함 블록의 백분율 (%)로 지정됨을 의미합니다.

이 요소의 height는 200px 이고 width는 50%입니다.

    div {
      height: 200px;
      width: 50%;
      background-color: powderblue;
    }

이 요소의 height는 100px 이고 width는 500px 입니다.

    div {
      height: 100px;
      width: 500px;
      background-color: powderblue;
    }

Note: height 및 width 속성은 패딩, 테두리, 또는 여백을 포함하지 않는다.

요소의 여백, 테두리 및 여백 안의 영역의 높이 / 너비를 설정합니다.

최대 너비 설정

이 max-width 속성은 요소의 최대 너비를 설정하는 데 사용됩니다.

이 max-width는 px, cm등과 같은 길이 값등 px, cm 원한다면 또는 %를 함유하는 블록들 또는 없음으로 설정

위의 문제 div는 브라우저 윈도우가 요소의 너비 보다 작을때 발생합니다. 그런 다음 브라우저는 가로 스크롤 막대를 페이지에 추가합니다.

max-width를 사용하는 대신, 이 상황에서 작은창 브라우저의 취급이 향상됩니다.

Tip: 브라우저 창을 500px너비 이하로 드래그하여 두 div의 차이점을 확인하십시오.

이 요소의 height는 100px 이고 최대 width는 500px 입니다.

참고: max-width 속성 값이 width를 재 정의합니다.

다음 예제는 div 높이가 100px 이고 최대 너비가 500px인 요소를 보여줍니다.

height = 100px max-width = 500px

    div {
      max-width: 500px;
      height: 100px;
      background-color: powderblue;
    }
