CSS Pseudo class

가상 클래스는 무엇인가?

가상 클래스는 요소의 특수 상태를 정의하는 데 사용된다.

예를 들어 다음과 같이 사용할 수 있다.

  • 사용자가 마우스를 올리면 요소의 스타일을 지정합니다.
  • 방문한 스타일과 방문하지 않은 스타일이 다르게 적용됩니다.
  • 포커스를 얻으면 요소의 스타일을 지정합니다.

syntax (문법)

    selector:pseudo-class {
      property:value;
    }

가상 클래스와 CSS 클래스

가상 클래스는 CSS 클래스와 결합할 수 있습니다.

    a.highlight:hover {
      color: #ff0000;
    }

div hover

hover속성에 css 속성을 정의하면 마우스를 올려놓았을시 그 css가 작동합니다.

    div:hover {
      background-color: yellow;
    }

간단한 툴팁 호버

div 요소위로 마우스를 가져가면 p 요소가 표시됩니다.

    p {
      display: none;
      background-color: yellow;
      padding: 20px;
    }
    div:hover p {
      display: block;
    }

CSS 첫번째 자식 가상 클래스

핵심: :first-child 의사 클래스는 다른 요소의 첫번쨰 자식을 지정합니다.

    p:first-child {
      color: blue;
    }

모든 p 요소의 첫 번째 i 요소와 일치시키기

다음 예제에서 선택기는 모든 p 요소의 첫 번쨰 i 요소를 찾습니다.

    p i:first-child {
      color: blue;
    }

모든 첫 번째 하위 p 요소의 모든 i 요소 일치

다음 예제에서 선택기는 다른 요소의 첫번째 자식인 p 요소의 모든 i 요소를 찾습니다.

    p:first-child i {
      color: blue;
    }
