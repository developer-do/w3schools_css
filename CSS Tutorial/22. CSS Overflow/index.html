CSS Layout - overflow

CSS overflow속성은 콘텐츠가 너무 커서 영역에 적합하지 않게되는 것을 제어합니다.

CSS overflow

이 overflow 속성은 요소의 내용이 너무 커서 지정한 영역에 들어가지 않을때 내용을 클립할지 또는 스크롤 막대를 추가할지 지정합니다.

이 overflow속성의 값은 다음과 같습니다.

  • visible - 기본값. overflow가 잘리지 않습니다. 그것은 요소의 상자 밖에 렌더링 됩니다.
  • hidden - overflow가 잘리고 나머지 내용은 보이지 않습니다.
  • scroll - overflow가 잘라지만 스크롤바가 추가되어 나머지 내용을 볼 수 있습니다.
  • auto - overflow가 잘린 경우 스크롤바를 추가하여 나머지 콘텐츠를 확인해야 합니다.

참고: 이 overflow 속성은 지정된 높이의 블록 요소에 대해서만 작동합니다.

참고: Mac에서는 OS X Lion에서 스크롤 막대는 기본적으로 숨겨져 있으며 "overflow : scroll"이 설정된 경우에도 사용시에 표시됩니다.

overflow: visible

기본적으로 overflow visible은 클리핑 되지 않고 요소 상자 외부로 렌터링 되는것을 의미합니다.

레이아웃을보다 잘 제어하고 싶을 때 overflow 속성을 사용할 수 있습니다. overflow 속성은 내용이 요소의 상자를 오버플로하면 어떻게되는지 지정합니다.

    p {
      width: 200px;
      height: 100px;
      background-color: #eee;
      overflow: visible;
    }

overflow: auto

이 auto값은 scroll과 유사합니다. 필요할 때만 스크롤 막대를 추가합니다.

overflow-x 및 overflow-y

overflow-x 및 overflow-y 특성은 수평 또는 수직 (또는 둘 다)의 콘텐츠를 플로우를 변경할지 여부를 지정합니다.

overflow-x 내용의 왼쪽 / 오른쪽 가장자리로 수행할 작업을 지정합니다.

overflow-y 내용의 위쪽 / 아래쪽 가장자리로 수행할 작업을 지정합니다.

    div {
      overflow-x: hidden; /* Hide horizontal scrollbar */
      overflow-y: scroll; /* Add vertical scrollbar */
    }
