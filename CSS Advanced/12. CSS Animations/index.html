CSS Animations

CSS animation을 사용하면 javascript 또는 flash를 사용하지 않고도 대부분의 HTML 요소만으로 애니메이션으로 만들 수 있습니다.

animation이란 무엇인가?

애니메이션을 사용하면 요소가 한 스타일에서 다른 스타일로 점진적으로 변경됩니다.

원하는 만큼 많은 CSS속성을 변경할 수 있습니다.

CSS 애니메이션을 사용하려면 먼저 애니메이션의 일쿠 keyframes를 지정해야합니다.

keyframes는 특정 시간에 요소의 스타일을 지정합니다.

@keyframes 규칙

@keyframes 규칙 내에서 CSS 스타일을 지정하면 애니메이션이 특정 시간에 현재 스타일에서 새 스타일로 점진적으로 변경됩니다.

애니메이션이 작동하도록 하려면 애니메이션을 요소에 바인딩 해야합니다.

다음 예제에서는 "예제" 애니메이션을 div 요소에 바이딩합니다. 애니메이션은 4초 동안 지속되며 div 요소의 배경색을 점차적으로 red에서 yellow로 변경합니다.

    @keyframes example {
      from { background-color: red;}
      to { background-color: yellow;}
    }

    #keyDiv {
      width: 100px;
      height: 100px;
      background-color: red;
      animation-name: example;
      animation-duration: 4s;
    }

참고: 이 animation-duration 속성은 애니메이션을 완료하는 데 걸리는 시간을 정의합니다. animation-duration 속성이 지정되지 않은 경우 기본값은 0으로 하기 때문에 어떤 애니메이션이든 발생하지 않습니다.

위의 예제에서 우리는 "from"과 "to"(0% ~ 100%를 나타냄) 키워드를 사용하여 스타일이 변경 될 때를 지정했습니다.

퍼센트를 사용할 수도 있습니다. 백분율을 사용하여 스타일 변경을 원하는 만큼 추가 할 수 있습니다.

다음 예제에서는 애니메이션이 25% 완료되고 50% 완료된 경우 및 애니메이션의 100% 완료시 다시 div요소의 배경색을 변경합니다.

    @keyframes example1 {
      0% {background-color: red;}
      25% {background-color: yellow;}
      50% {background-color: blue;}
      100% {background-color: green;}
    }

    #keyDiv1 {
      width: 100px;
      height: 100px;
      background-color: red;
      animation-name: example1;
      animation-duration: 4s;
    }

다음 예제는 애니메이션이 25%완료되고 50%완료 될때 및 애니메이션이 100% 완료될 때마다 div의 배경색과 위치를 바꿔줍니다.

    @keyframes example2 {
      0% { background-color: red; left: 0px; top: 0px;}
      25% { background-color: yellow; left: 200px; top: 0px;}
      50% { background-color: blue; left: 200px; top: 200px;}
      75% { background-color: green; left: 0px; top: 200px;}
      100% { background-color: red; left: 0px; top: 0px;}
    }
    #keyDiv2 {
      width: 100px;
      height: 100px;
      position: relative;
      background-color: red;
      animation-name: example2;
      animation-duration: 4s;
    }

animation-delay

이 animation-delay 속성은 애니메이션의 시작에 대한 지연을 지정합니다.

다음 예제는 애니메이션을 시작하기 전에 2초의 지연이 있습니다.

    #keyDiv3 {
      width: 100px;
      height: 100px;
      position: relative;
      background-color: red;
      animation-name: example2;
      animation-duration: 4s;
      animation-delay: 2s;
    }

음수값도 허용됩니다. 음수값을 사용하면 이미 N초 동안 재생 된 것처럼 애니메이션이 시작됩니다.

다음 예제에서는 애니메이션이 이미 2초동안 재생된 것처럼 시작됩니다.

    #keyDiv4 {
      width: 100px;
      height: 100px;
      position: relative;
      background-color: red;
      animation-name: example2;
      animation-duration: 4s;
      animation-delay: -2s;
    }

애니메이션을 몇 번 실행해야하는지 설정

이 animation-iteration-count 속성은 애니메이션이 실행되어야 하는 횟수를 지정합니다.

다음 예제는 정지되기 전에 애니메이션을 세 번 실행합니다.

    #keyDiv5 {
      width: 100px;
      height: 100px;
      position: relative;
      background-color: red;
      animation-name: example2;
      animation-duration: 4s;
      animation-iteration-count: 3;
    }

다음 예제에서는 infinite 값을 사용하여 애니메이션을 영원히 계속 만듭니다.

    #keyDiv6 {
      width: 100px;
      height: 100px;
      position: relative;
      background-color: red;
      animation-name: example2;
      animation-duration: 4s;
      animation-iteration-count: infinite;
    }

애니메이션을 역방향 또는 대체사이클로 실행하십시오.

이 animation-direction 속성은 애니메이션을 앞으로, 뒤로 또는 번갈아 재상해야하는지 여부를 지정합니다.

애니메이션 방향 속성의 값은 다음과 같습니다.

  • normal - 애니메이션이 정상적으로 재생됩니다.
  • reverse - 애니메이션이 역방향으로 재생됩니다.
  • alternate - 애니메이션이 먼저 재생된 후 뒤로 재생됩니다.
  • alternate-reverse - 애니메이션이 먼저 뒤로 재생 된 다음 앞으로 재생됩니다.

다음 예제에서는 역방향으로 애니메이션을 실행합니다.

    #keyDiv7 {
      width: 100px;
      height: 100px;
      position: relative;
      background-color: red;
      animation-name: example2;
      animation-duration: 4s;
      animation-direction: reverse;
    }

다음 예제에서는 "alternate" 값을 사용하여 애니메이션이 먼저 앞으로 이동한 다음 뒤로 이동합니다.

    #keyDiv8 {
      width: 100px;
      height: 100px;
      position: relative;
      background-color: red;
      animation-name: example2;
      animation-duration: 4s;
      animation-iteration-count: 2;
      animation-direction: alternate;
    }

다음 예제에서는 "alternate-reverse" 값을 사용하여 애니메이션을 먼저 뒤로 실행한 다음 전달합니다.

    #keyDiv9 {
      width: 100px;
      height: 100px;
      position: relative;
      background-color: red;
      animation-name: example2;
      animation-duration: 4s;
      animation-iteration-count: 2;
      animation-direction: alternate-reverse;
    }

animation-timing 지정

이 animation-timing-function 속성은 애니메이션의 속도를 지정합니다.

animation-timing-function 속성은 다음 값을 가질 수 있습니다.

  • ease - 느린 시작, 중간 빠름 마지막 느리게 끝나는 애니메이션
  • linear - 시작부터 끝까지 동일한속도로 애니메이션을 지정합니다.
  • ease-in - 느린 시작으로 애니메이션을 지정합니다.
  • ease-out - 끝이 느린 애니메이션을 지정합니다.
  • ease-in-out - 시작과 끝이 느린 애니메이션을 지정합니다.
  • cubic-bezier(n,n,n,n) - cubic-bezier함수로 자신만의 값을 정의할 수 있습니다.

애니메이션의 채우기 모드 지정

CSS 애니메이션은 첫 번쨰 키 프레임이 재생되거나 마지막 키 프레임이 재생된 후에 요소에 영향을 주지 않습니다.

이 animation-fill-mode 속성은애니메이션이 재생되지 않을 때 (시작 전, 끝나기 전 또는 끝날 때)에 대상 요소의 스타일을 지정합니다.

aniumation-fill-mode 속성은 다음 값을 가질 수 있습니다.

  • none - 기본값. 애니메이션은 실행되기 전후에 요소에 스타일을 적용하지 않습니다.
  • forwards - 요소는 마지막 키 프레임에서 설정 한스타일 값을유지합니다. (애니메이션 방향 및 애니메이션 반복횟수에 따라 다름)
  • backwards - 요소는 첫 번쨰 키프레임에 의해 설정된 스타일 값을 가져오고 (애니메이션 방향에 따라 다름) 애니메이션 지연 기간 동안 이 값을 유지합니다.
  • both - 애니메이션 앞 뒤 모두에 대한 규칙을 따르고 양방향으로 애니메이션 속성을 확장합니다.

다음 예제는 애니메이션이 끝날 때 div 요소가 마지막 키 프레임의 스타일 값을 유지하도록 합니다.

    #keyDiv10 {
      width: 100px;
      height: 100px;
      background-color: red;
      position: relative;
      animation-name: example3;
      animation-duration: 3s;
      animation-fill-mode: forwards;
    }

다음 예제에서는 div 요소가 애니메이션이 시작되지 전에 첫번쨰 키프레임에 의해 설정된 스타일 값을 가져옵니다.

    #keyDiv11 {
      width: 100px;
      height: 100px;
      background-color: red;
      position: relative;
      animation-name: example3;
      animation-duration: 3s;
      animation-fill-mode: backwards;
    }

다음 예제에서는 div 요소가 애니메이션이 시작되기 전에 첫 번째 키 프레임에 의해 설정된 스타일 값을 가져오고 애니메이션이 끝나면 마지막 키 프레임에서 스타일 값을 유지할 수 있습니다.

    #keyDiv12 {
      width: 100px;
      height: 100px;
      background: red;
      position: relative;
      animation-name: example;
      animation-duration: 3s;
      animation-fill-mode: both;
    }

애니메이션 속기 속성

아래 예제는 6가지 애니메이션 속성을 사용합니다.

    div {
      animation-name: example;
      animation-duration: 5s;
      animation-timing-function: linear;
      animation-delay: 2s;
      animation-iteration-count: infinite;
      animation-direction: alternate;
    }

위와 같은 코드를 속기속성으로 쓰면

    div {
      animation: example 5s linear 2s infinite alternate;
    }

위와같이 쓸 수 있습니다.
