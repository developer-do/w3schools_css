CSS 2D transform

CSS 변환

CSS 변환을 사용하면 요소를 변환, 회전, 크기 조절 및 삐뚤어지게 할 수 있습니다.

변형은 element가 모양, 크기 및 위치를 변경하게 하는 효과입니다.

CSS는 2D 및 3D 변환을 지원합니다.

CSS 2D 변환

이 장에서는 다음 2D 변환 방법에 대해 학습합니다.

  • translate()
  • rotate()
  • scale()
  • skewX()
  • skewY()
  • matrix()

translate() 메서드

이 translate() 메서드는 요소를 현재 위치에서 X 축 및 Y 축에 지정된 매개 변수에 따라 이동합니다.

다음 예제에서는 div 요소를 오른쪽으로 50px 이동하고 현재 위치에서 100px 아래로 이동합니다.

    div {
      -ms-transform: translate(50px, 100px);
      -webkit-transform: translate(50px, 100px);
      transform: translate(50px, 100px);
    }

rotate() 메서드

이 rotate() 메소드는 주어진 정도에 따라 element를 시계 방향 또는 시계 반대 방향으로 회전합니다.

다음 예제에서는 div element를 시계 방향으로 20도 회전합니다.(음수는 시계 반대방향)

    div {
      -ms-transform: rotate(20deg); /* IE 9 */
      -webkit-transform: rotate(20deg); /* Safari */
      transform: rotate(20deg);
    }

scale() 메서드

이 scale() 메서드는 element의 크기를 늘리거나 줄입니다.(폭 및 높이에 대한 매개변수에 따라.)

다음 예제에서는 div element를 원래 width의 2배, height의 3배로 늘립니다.

    div {
      -ms-transform: scale(2, 3); /* IE 9 */
      -webkit-transform: scale(2, 3); /* Safari */
      transform: scale(2, 3);
    }

다음 예제는 div element를 원래 width, height를 반으로 줄입니다.

    div {
      -ms-transform: scale(0.5, 0.5); /* IE 9 */
      -webkit-transform: scale(0.5, 0.5); /* Safari */
      transform: scale(0.5, 0.5);
    }

skewX() 메서드

이 skewX() 메서드는 X 축을 따라 지정된 각도로 요소를 기울입니다.

다음 예제에서는 X 축을따라 20도 div element을 기울입니다.

    div {
      -ms-transform: skewX(20deg); /* IE 9 */
      -webkit-transform: skewX(20deg); /* Safari */
      transform: skewX(20deg);
    }

skewY() 메서드

이 skewY() 메서드는 Y 축을 따라 지정된 각도로 요소를 기울입니다.

다음 예제에서는 Y 축을따라 20도 div element을 기울입니다.

    div {
      -ms-transform: skewY(20deg); /* IE 9 */
      -webkit-transform: skewY(20deg); /* Safari */
      transform: skewY(20deg);
    }

skew() 메서드

이 skew() 메서드는 X 및 Y 축을 따라 지정된 각도만큼 element을 기울입니다.

다음 예제에서는 div element을 X 축을따라 20도 기울이고 Y 축을따라 10도 기울입니다.

    div {
      -ms-transform: skew(20deg, 10deg); /* IE 9 */
      -webkit-transform: skew(20deg, 10deg); /* Safari */
      transform: skew(20deg, 10deg);
    }

참고: 두개의 매개변수를 지정하지 않고 한개만 지정했을 경우에는 x 축값만 바뀐다.

matrix() 메서드

이 matrix() 방법은 모든 2D 변환 방법을 하나로 결합합니다.

matrix() 메서드는 수학 함수가 포함 된 6개의 매개 변수를 사용하여 요소를 회전, 크기 조절, 이동 (병합) 및 기울이기를 허용합니다.

매개변수는 다음과 같습니다. matrix(scaleX(), skewY(), skewX(), scaleY(), translateX(), translateY())

    div {
      -ms-transform: matrix(1, -0.3, 0, 1, 0, 0); /* IE 9 */
      -webkit-transform: matrix(1, -0.3, 0, 1, 0, 0); /* Safari */
      transform: matrix(1, -0.3, 0, 1, 0, 0);
    }
