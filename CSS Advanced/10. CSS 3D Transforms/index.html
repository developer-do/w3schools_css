CSS 3D Transforms

CSS 3D 변환

CSS를 사용하면 3D 변형을 사용하여 요소의 서식을 지정할 수 있습니다.

CSS 3D 변환

이 장에서는 다음 3D 변형 방법에 대해 알아봅니다.

  • rotateX()
  • rotateY()
  • rotateZ()

rotateX() 메서드

이 rotateX() 방법은 주어진 각도에서 X축을 중심으로 요소를 회전시킵니다.

dkdkdkdkdkdk
    #myDiv {
      -webkit-transform: rotateX(150deg);
      transform: rotateX(150deg);
    }

rotateY() 메서드

이 rotateY() 메소드는 주어진 각도에서 Y 축 중심으로 요소를 회전시킵니다.

dkdkdkdkdkdk
    #myDiv1 {
      -webkit-transform: rotateY(130deg);
      transform: rotateY(130deg);
    }

rotateZ() 메서드

이 rotateZ() 방법은 주어진 각도에서 Z 축을 중심으로 요소를 회전시킵니다.

dkdkdkdkdkk
    #myDiv2 {
      -webkit-transform: rotateZ(90deg);
      transform: rotateZ(90deg);
    }
